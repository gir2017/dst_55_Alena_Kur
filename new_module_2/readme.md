# Project: HH vacancy analysis
##### Done by Alena Kurylchyk


## Project objective ##
Need to analyze vacancies in the IT field from the Headhunter website using SQL database.


## What has been done ##
1. Getting to know the data: the number of vacancies in the database, the number of employers, and so on.
2. Analysis of vacancies: distribution of the number of vacancies by region, by requested experience, salary data, and so on.
3. Analysis of employers: which companies offer the most vacancies and the best salaries, in which regions, and so on.
4. Analysis of vacancies in the field of Data Science: number of such vacancies, expected work experience, required skills, etc.


## Results ##
+ Conclusions about the available data in the context of employing data scientists and suggest options for further research.
